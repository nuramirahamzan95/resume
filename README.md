# Modern Full-Stack Developer Portfolio — Nuramirah Hamzan

A fresh, modern, and highly responsive single-page portfolio website designed for a **Full-Stack Software Developer**. Built with a minimalist tech-centric dark mode, subtle background ambient glow effects, dynamic bento-grid components, and sleek glassmorphism card layouts.

The content is fully updated to highlight enterprise production experience, specialized application engineering, and database optimization.

---

## 🚀 Key Features

*   **Premium Dark Aesthetic:** Clean, high-contrast UI tailored for modern software engineering professionals.
*   **Bento Grid Layout:** Clean categorical display showcasing Core Tech Stacks, databases, tools, and workflows at a single glance.
*   **Fully Responsive Infrastructure:** Built using utility-first fluid classes that transition seamlessly from desktop screens down to mobile devices.
*   **SEO & Performance Focused:** Built entirely with lightweight vanilla HTML5, asynchronous icon fonts, and compiled CDN setups without unnecessary legacy dependencies like Bootstrap.

---

## 🛠️ Core Architecture & Tech Stack

*   **Frontend UI Framework:** Tailwind CSS (Utility-First Design Engine)
*   **Dynamic Client Ecosystem:** Vue.js, Inertia.js, Alpine-ready semantics
*   **Backend Engineering Focus:** Laravel, PHP, RESTful API Design
*   **Database Infrastructure:** MySQL, PostgreSQL
*   **Operations & Tools:** Git / GitLab CI/CD, Jira, Linux/Windows Environment Scripting
*   **Design Tokens:** Inter Font Family, Remix Icons Vector Toolkit
